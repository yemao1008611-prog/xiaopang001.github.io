语音合成 (TTS)

edge-tts 语音合成，支持多种中文音色选择

应用场景

  • 文字转语音播报
  • 有声内容生成
  • 语音提醒播报
  • 多语言语音合成
Python 
import edge_tts
import asyncio

async def tts():
    communicate = edge_tts.Communicate("你好，这是语音合成", "zh-CN-XiaoxiaoNeural")
    await communicate.save("output.mp3")

asyncio.run(tts())
