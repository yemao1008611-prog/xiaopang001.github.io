语音识别 (Whisper)

基于 OpenAI Whisper 的离线语音识别，支持多语言高精度转录

应用场景

  • 飞书语音消息转文字
  • 会议录音自动转写
  • 音频文件批量转录
  • 实时语音流识别
Shell 
# 基础转录（使用默认 base 模型）
~/.clawdbot/skills/local-whisper/scripts/local-whisper audio.wav

# 使用更好的模型（turbo 更快更好）
~/.clawdbot/skills/local-whisper/scripts/local-whisper audio.wav --model turbo

# 带时间戳输出
~/.clawdbot/skills/local-whisper/scripts/local-whisper audio.wav --timestamps

# 输出 JSON 格式
~/.clawdbot/skills/local-whisper/scripts/local-whisper audio.wav --model turbo --json

# 指定语言（中文）
~/.clawdbot/skills/local-whisper/scripts/local-whisper audio.wav --language zh

# 完整示例：转录 + 保存结果
~/.clawdbot/skills/local-whisper/scripts/local-whisper meeting.wav --model turbo --timestamps --output result.txt
Setup 
# 安装依赖（需要 ffmpeg）
# Ubuntu/Debian
sudo apt install ffmpeg

# macOS
brew install ffmpeg

# 安装 Whisper 技能
clawhub install local-whisper

# 或手动安装
cd ~/.clawdbot/skills/local-whisper
uv venv .venv --python 3.12
uv pip install --python .venv/bin/python click openai-whisper torch

语音合成 (TTS)

基于 Edge TTS 的语音合成，支持多种中文音色

应用场景

  • 文字转语音播报
  • 有声内容生成
  • 语音提醒播报
  • 多语言语音合成
Shell 
# 使用 edge-tts 语音合成
edge-tts --voice "zh-CN-YunxiNeural" --text "你好，我是小胖" --write-media /tmp/voice.mp3

# 使用不同音色
edge-tts --voice "zh-CN-XiaoxiaoNeural" --text "你好" --write-media /tmp/xiaoxiao.mp3  # 女声
edge-tts --voice "zh-CN-YunyangNeural" --text "你好" --write-media /tmp/yunyang.mp3    # 男声

# 完整示例：生成播报
edge-tts --voice "zh-CN-YunxiNeural" \
  --text "今天是2026年3月14日，星期日。今天的工作任务是..." \
  --write-media /tmp/daily_report.mp3

# 批量转换（将文本文件转为语音）
while read line; do
  edge-tts --voice "zh-CN-YunxiNeural" --text "$line" --write-media "audio_$i.mp3"
done < text_file.txt

图像理解

基于视觉大模型的图像理解与分析

应用场景

  • 截图分析理解
  • 图表数据提取
  • UI界面分析
  • 文档扫描理解
OpenClaw Tool 
# 分析图片内容（通过 OpenClaw image 工具）
# 直接描述图片中的内容

image image="/path/to/image.jpg" \
  prompt="请详细描述这张图片的内容"

# 分析截图
image image="https://example.com/screenshot.png" \
  prompt="这个页面的布局是怎样的？有哪些主要元素？"

# 分析图表
image image="chart.png" \
  prompt="这个图表展示什么数据？请提取关键数值"

# 分析代码截图
image image="code.png" \
  prompt="这段代码实现什么功能？"

内容总结

对长文本、PDF、网页等内容进行智能总结

应用场景

  • 长文章摘要
  • PDF文档要点提取
  • 网页内容凝练
  • 会议纪要生成
Tools 
# 总结 URL 内容
web_fetch url="https://example.com/article" maxChars=10000

# 分析 PDF 文档
pdf pdf="/path/to/document.pdf" pages="1-5" prompt="提取文档核心要点"

# 总结图片（带文字）
image image="document.jpg" prompt="提取图片中所有文字内容并总结要点"

# 批量总结多个文件
pdf pdfs='["doc1.pdf","doc2.pdf"]' prompt="对比分析这两个文档的异同"
