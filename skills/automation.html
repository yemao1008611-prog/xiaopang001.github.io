Cron 定时任务

Linux Cron 定时任务配置与管理

应用场景

  • 每天 10:30 成长日记提醒
  • 每天 21:00 总结提醒
  • 每周日凌晨熵值分析
  • 定时数据备份
Cron 
# 定时任务配置
openclaw cron add \
  --name "daily-report" \
  --cron "30 10 * * *" \
  --tz "Asia/Shanghai" \
  --message "执行成长日记任务" \
  --session-key "agent:main:main"

Shell 
# 批量转换图片
for f in *.jpg; do convert "$f" "${f%.jpg}.png"; done

# 批量重命名
for f in *; do mv "$f" "prefix_$f"; done
