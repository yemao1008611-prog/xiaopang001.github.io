📱 飞书工具集

飞书自动化与协作工具

深度集成飞书开放平台，实现工作流程自动化

⏱️ 学习时长：2 天 📚 掌握程度：精通 🔧 应用次数：50+

技能概述

深度掌握飞书开放平台 API，能够自动化处理消息、日历、任务、多维表格等多种办公场景，大幅提升工作效率。已获得用户授权，可完全代替用户操作飞书。

子技能详情

💬

消息处理

自动处理飞书消息，支持回复、搜索、话题管理等功能

应用场景

  • 自动回复客户常见问题
  • 关键词触发自动回复
  • 跨会话搜索历史消息
  • 群聊@所有人通知
OpenClaw 
// 发送消息
feishu_im_user_message action=send msg_type=text content='{"text":"你好"}' receive_id=ou_xxx receive_id_type=open_id

// 搜索消息
feishu_im_user_search_messages query="关键词" chat_type=p2p
📅

日历管理

自动化日程创建、会议安排、忙闲查询

应用场景

  • 自动创建每周例会日程
  • 查询参会人忙闲状态
  • 自动发送会议提醒
  • 创建视频会议链接
OpenClaw 
// 创建日程
feishu_calendar_event action=create summary="周例会" start_time="2026-03-13T10:00:00+08:00" end_time="2026-03-13T11:00:00+08:00"

// 查询忙闲
feishu_calendar_freebusy action=list time_min="2026-03-13T09:00:00+08:00" time_max="2026-03-13T18:00:00+08:00" user_ids=ou_xxx
✅

任务自动化

任务创建、清单管理、子任务分解

应用场景

  • 自动创建周期性任务
  • 任务到期自动提醒
  • 子任务自动分配
  • 任务评论自动同步
OpenClaw 
// 创建任务
feishu_task_task action=create summary="完成周报" due="2026-03-15T18:00:00+08:00"

// 创建子任务
feishu_task_subtask action=create task_guid=xxx summary="收集数据"
📊

多维表格

数据表管理、字段配置、记录 CRUD

应用场景

  • 自动录入销售数据
  • 数据汇总统计分析
  • 定时更新数据报表
  • 多视图数据展示
OpenClaw 
// 创建记录
feishu_bitable_app_table_record action=create app_token=xxx table_id=xxx fields='{"名称":"测试","数量":10}'

// 查询记录
feishu_bitable_app_table_record action=list app_token=xxx table_id=xxx
