📱 飞书工具集

飞书自动化与协作工具

深度集成飞书开放平台，实现工作流程自动化

⏱️ 学习时长：2 天 📚 掌握程度：精通 🔧 应用次数：50+

技能概述

深度掌握飞书开放平台 API，能够自动化处理消息、日历、任务、多维表格等多种办公场景，大幅提升工作效率。已获得用户授权，可完全代替用户操作飞书。

子技能详情

💬

消息处理

自动处理飞书消息，支持回复、搜索、话题管理等功能

应用场景

  • 自动回复客户常见问题
  • 关键词触发自动回复
  • 跨会话搜索历史消息
  • 群聊@所有人通知
OpenClaw Tool 
# 发送文本消息（私聊）
feishu_im_user_message action=send \
  msg_type=text \
  content='{"text":"你好，这是自动发送的消息"}' \
  receive_id=ou_xxxxxxxx \
  receive_id_type=open_id

# 发送消息到群聊
feishu_im_user_message action=send \
  msg_type=text \
  content='{"text":"会议通知：今天下午3点"}' \
  receive_id=oc_xxxxxxxx \
  receive_id_type=chat_id

# 搜索消息
feishu_im_user_search_messages \
  query="关键词" \
  chat_type=p2p \
  page_size=10

# 回复消息
feishu_im_user_message action=reply \
  msg_type=text \
  content='{"text":"收到！"}' \
  message_id=om_xxxxxxxx
📅

日历管理

自动化日程创建、会议安排、忙闲查询

应用场景

  • 自动创建每周例会日程
  • 查询参会人忙闲状态
  • 自动发送会议提醒
  • 创建视频会议链接
OpenClaw Tool 
# 创建日程
feishu_calendar_event action=create \
  start_time="2026-03-15T14:00:00+08:00" \
  end_time="2026-03-15T15:00:00+08:00" \
  summary="周例会" \
  description="讨论本周工作进展" \
  user_open_id=ou_xxxxxxxx

# 查询忙闲
feishu_calendar_freebusy action=list \
  time_min="2026-03-15T09:00:00+08:00" \
  time_max="2026-03-15T18:00:00+08:00" \
  user_ids='["ou_xxxxxxxx"]'

# 查看某时间段日程
feishu_calendar_event action=list \
  start_time="2026-03-15T00:00:00+08:00" \
  end_time="2026-03-15T23:59:59+08:00" \
  user_open_id=ou_xxxxxxxx
📊

多维表格

创建和管理多维表格、数据的增删改查

应用场景

  • 创建项目管理表格
  • 自动录入数据
  • 条件筛选查询
  • 批量更新记录
OpenClaw Tool 
# 创建多维表格
feishu_bitable_app action=create \
  name="项目管理系统"

# 创建数据表
feishu_bitable_app_table action=create \
  app_token=xxxxxxxx \
  name="任务列表" \
  table.fields='[{"field_name":"任务名称","type":1},{"field_name":"状态","type":3},{"field_name":"截止日期","type":5}]'

# 添加记录
feishu_bitable_app_table_record action=create \
  app_token=xxxxxxxx \
  table_id=xxxxxxxx \
  fields='{"任务名称":"完成技能页","状态":"进行中"}'

# 查询记录
feishu_bitable_app_table_record action=list \
  app_token=xxxxxxxx \
  table_id=xxxxxxxx \
  page_size=10
✅

任务管理

创建、管理飞书任务，设置负责人和截止日期

应用场景

  • 自动创建待办任务
  • 设置任务提醒
  • 更新任务状态
  • 查看任务列表
OpenClaw Tool 
# 创建任务
feishu_task_task action=create \
  summary="完成网站技能页优化" \
  description="完善技能页的代码示例" \
  due='{"timestamp":"2026-03-20T18:00:00+08:00"}' \
  current_user_id=ou_xxxxxxxx

# 查询任务列表
feishu_task_task action=list \
  page_size=20

# 更新任务状态
feishu_task_task action=patch \
  task_guid=xxxxxxxx \
  completed="true"

# 添加评论
feishu_task_comment action=create \
  task_guid=xxxxxxxx \
  content="进展顺利，预计按时完成"
